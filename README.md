Business Rule nedir, ne ucun istifade olunur

Business Rule nədir?

Business Rule — sistemin necə davranmalı olduğunu müəyyən edən biznes qaydasıdır.

Yəni proqramın texniki işləmə mexanizmi yox, şirkətin / layihənin qoyduğu qaydalar.

Sadə dillə:

“Bu sistemdə hansı şərtlə nə etmək olar?” sualının cavabı.

Nə üçün istifadə olunur?

1️⃣ Sistemin məntiqini qorumaq üçün
Proqram istənilən əməliyyatı etməməlidir. Qayda olmalıdır.

2️⃣ Yanlış əməliyyatların qarşısını almaq üçün
Məsələn:

Eyni email ilə iki dəfə qeydiyyat olmaz

Mənfi balansla alış etmək olmaz

User başqasının məlumatını dəyişə bilməz

3️⃣ Biznes tələblərini kod səviyyəsində tətbiq etmək üçün
Şirkət hansı qaydanı qoyubsa, proqram da onu tətbiq edir.

4️⃣ Security və icazə mexanizmini idarə etmək üçün
Kim nə edə bilər — bunu müəyyən edir.
